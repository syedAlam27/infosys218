TERMINAL

pwd - current directory ls - list contents for current folder cd - change directory mkdir - make new directory mv - move and copy file touch - new file vi - command/insert

GIT

git init - initialize git git add ... ("." includes all files) - add files to local repo git commit -m ... - commits files to remote git checkout (-b) (-b makes new branch and switches to it) - navigate between branches git push - push files to remote repo

Docker

docker compose up - start or restart all the services defined in a docker-compose.yml. docker compose down - stops and removes container Docker ps - lists running containers And docker kill - shuts the container off
